NAME
          pnet_lo - Get lower bound for local sublattice

     SYNOPSIS
          #include <mpi.h>
          #include "pnet.h"

          int pnet_lo(const Pnet *pnet);

     DESCRIPTION
          Function returns lower bound for local sublattice. Function
          is implemented as a macro.  The argument of pnet_lo is:

          pnet Pointer to initialized Pnet struct

     RETURN VALUE
          Upon successful completion lower bound is returned.

     AUTHOR
          Werner von Bloh (bloh@pik-potsdam.de)
