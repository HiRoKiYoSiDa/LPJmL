     NAME
          getcountry - extract grid cells for specified countries

     SYNOPSIS
          getcountry [-list] [-longheader] countryfile gridfile
          outfile countrycode ...

     DESCRIPTION
          Program extracts country grid cells from grid file.

     OPTIONS
          -list
               print list of country codes defined

          -longheader
               set long header in grid and country code file

          countryfile
               country code file

          gridfile
               filename of binary grid data file

          outfile
               filename of grid data file written

          countrycode
               ISO 3166-1 alpha-3 abbreviations for countries. or
               country codes as numbers defined in include/managepar.h

     EXAMPLES
          Create grid data file for USA:
               getcountry grid.bin grid_usa.clm USA

     EXIT STATUS
          getcountry returns a zero exit status if the grid file is
          created.  Non zero is returned in case of failure.
