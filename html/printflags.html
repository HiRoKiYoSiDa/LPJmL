NAME
          printflags - Print LPJmL compilation flags and compiler used

     SYNOPSIS
          #include "lpj.h"

          void printflags(void);


     DESCRIPTION
          Function print LPJmL compilation flags, compiler used,
          endianness and 32/64 bit mode.

     RETURN VALUE
          None.
