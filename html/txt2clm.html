     NAME
          txt2clm - convert text files to clm data files for LPJmL

     SYNOPSIS
          txt2clm [-h] [-float] [-int] [-scale s] [-nbands n]
          [-firstyear f] [-firstcell c] [-ncell n]  [-header id]
          txtfile clmfile

     DESCRIPTION
          Program converts text files containing white space separated
          numbers into LPJ climate data files. File is read until end
          of file is reached.

     OPTIONS
          -h   display a short help text

          -float
               write data as float, default is short

          -int write data as int, default is short

          -scale s
               scale data by a factor

          -nbands n
               set number of bands, default is 12

          -firstyear f
               set first year, default is 1901

          -firstcell c
               set first cell, default is 0

          -ncell n
               set number of cells, default is 67420

          -header id
               set header string in clm file, default is LPJCLIM

          txtfile
               filename of text file

          clmfile
               filename of clm data file written

     EXAMPLES
          Create clm data from text file:
               txt2clm -nbands 365 tmp_1901-2003.txt tmp2003.clm

     EXIT STATUS
          txt2clm returns a zero exit status if the clm file is
          created.  Non zero is returned in case of failure.
