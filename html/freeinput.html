NAME
          freeinput - Closes files and deallocates input data

     SYNOPSIS
          #include "lpj.h"

          void freeinput(Input input);


     DESCRIPTION
          Function closes open files and deallocates input data.  The
          argument of freeinput is:

          input
               Pointer to input data.  Must be initialized by
               initinput().

     RETURN VALUE
          None.
