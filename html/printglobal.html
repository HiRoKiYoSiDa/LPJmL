     NAME
          printglobal - print global values for raw binary output
          files

     SYNOPSIS
          printglobal [-firstyear year] [-nbands n] [-cellsize
          gridfile  outfile ...

     DESCRIPTION
          Program prints global area weighted sums or means of LPJmL
          binary output files.

     OPTIONS
          -firstyear year
               Set first year in output, default is 1901

          -nbands n
               Set number of bands, default is 1

          -cellsize size
               Set size of cell in degrees, default is 0.5 degree

          -day|month
               Set time step of input file to day or month, default is
               year

          -noheader
               Do not print header for columns

          -csv Set ',' as separator in output, default is space

          -mean
               Print global mean value instead of global sum

          -yearsum
               Print yearly sum of monthly or daily data

          gridfile
               Filename of binary grid file

          outfile
               Filename(s) of raw binary output files

     EXAMPLE
          Print global annual sum of NPP and heterotrophic respiration:
               printglobal -month -yearsum grid.bin mnpp.bin mrh.bin

     EXIT STATUS
          printglobal returns a zero exit status if the output files
          can be read.  Non zero is returned in case of failure.
