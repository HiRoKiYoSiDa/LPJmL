     NAME
          country2cdf - convert country/region code file into NetCDF
          file

     SYNOPSIS
          country2cdf [-global] [-varname gridfile countryfile
          netcdffile

     DESCRIPTION
          Program converts country/region input file into NetCDF file.

     OPTIONS
          -global
               use global grid for NetCDF file

          -cellsize size
               set cell size in grid file to size

          -compress level
               compression level for NetCDF4 files

          -descr d
               set long name in NetCDF file

          -index {0|1}
               set country  (0) or region (1) in file. Default is 0

          varname
               variable name in NetCDF file

          gridfile
               filename of grid data file

          countryfile
               filename of country/region code data file

          netcdffile
               filename of NetCDF file created

     EXAMPLES
          Create data file for country and region:
               country2cdf country  grid.bin cow.bin country.nc
               country2cdf [-index 1] region  grid.bin cow.bin
               region.nc

     EXIT STATUS
          country2cdf returns a zero exit status if the NetCDF file is
          created.  Non zero is returned in case of failure.
