     NAME
          getcellindex - get cell index for coordinates in grid clm
          file

     SYNOPSIS
          getcellindex [-search] [-longheader] gridfile [lat[N|S]
          lon[E|W]] ...]

     DESCRIPTION
          Program gets cell index for coordinates in grid file. If no
          cell coordinates are provided, all coordinates in the grid
          file are listed.

     OPTIONS
          -search
               search for nearest coordinate in grid file if
               coordinate was not found

          gridfile
               filename of binary grid data file in CLM format

          lat  latitude in degree

          lon  longitude in degree

     EXAMPLES
          Get cell index for 48.25 12.25:
               getcountry grid.clm 48.25 12.25

     EXIT STATUS
          Non zero is returned in case of failure.
