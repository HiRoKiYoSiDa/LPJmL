     NAME
          mergeclm - merge LPJmL climate data files

     SYNOPSIS
          mergeclm [-f] [-longheader] infile ... outfile

     DESCRIPTION
          Program merges bands of climate data files in clm format
          into one clm file.

     OPTIONS
          -f   Force overwriting of extisting output file

          -longheader
               Long (version 2) header assumed for clm files infile
               Filename(s) of clm files to paste. If filename is
               "zero" one band containing all zero values is added

          outfile
               Filename of merged clm file

     EXAMPLE
          Merge clm data file from 12 clm files:
               mergeclm temp_1901-2003_01.clm temp_1901-2003_02.clm
               temp_1901-2003_03.clm temp_1901-2003_04.clm temp_1901-
               2003_05.clm

     EXIT STATUS
          mergeclm returns a zero exit status if the clm file is
          created.  Non zero is returned in case of failure.
