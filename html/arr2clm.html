     NAME
          arr2clm - convert 2-D float arrays into CLM file

     SYNOPSIS
          arr2clm [-firstyear y] [-scale factor] [-id string] [-float]
          [-version v] [-r value] [-invalid value] gridfile arrfile
          clmfile

     DESCRIPTION
          Program converts binary 2-D float arrays into CLM input
          data. The array has to be of size 360/lon_resolution x
          180/lat_resolution. The latitudes have to start with 90-
          0.5*lat_resolution, the longitudes with
          -180+0.5*lon_resolution. The reolution is taken from the
          grid file.

     OPTIONS
          -firstyear y
               Set first year in CLM file, default is 1901

          -scale factor
               scaling factor for CLM files. Default is one

          -id string
               LPJ header string in clm file

          -version v
               version of clm header, default is 3

          -float
               write float values in clm file, default is short

          -invalid value
               value of invalid data in array file, default is -9999.

          -r value
               value to replace invalid data in CLM file, default is
               the invalid value.

          gridfile
               filename of grid data file

          arrfile
               filename of array file to convert

          clmfile
               filename of CLM file

     EXAMPLES
          Create CLMata file replacing invalid data by 10.
               arr2clm -float -r 10.0 grid.clm array.bin data.clm

     EXIT STATUS
          arr2clm returns a zero exit status if the CLM file is
          created.  Non zero is returned in case of failure.
