     NAME
          cdf2clm - convert NetCDF files into CLM files

     SYNOPSIS
          cdf2clm [-scale factor] [-units u] [-var name] [-o clmfile]
          gridfile clmfile ...

     DESCRIPTION
          Program converts NetCDF input data into CLM input data.

     OPTIONS
          -scale factor
               scaling factor for CLM files. Default is one -units u
               set units in NetCDF file

          -var name
               variable name in NetCDF file

          -o clmfile
               filename of CLM data file written. Default is out.clm

          gridfile
               filename of grid data file

          netcdffile
               filename of NetCDF file(s) converted

     EXAMPLES
          Create data file for temperature:
               cdf2clm -units celsius -var tas -o temp1901-2010.clm
               grid.clm tmp*.nc

     EXIT STATUS
          cdf2clm returns a zero exit status if the CLM file is
          created.  Non zero is returned in case of failure.
